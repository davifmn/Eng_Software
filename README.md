## Objetivo do Sistema
O BarberFlow é uma secretaria virtual inteligente projetada para automatizar o agendamento em barbearias. O sistema permite que clientes visualizem serviços, profissionais disponíveis e horários em tempo real, eliminando a necessidade de atendimento humano para marcações simples. O foco é proporcionar uma experiência fluida no navegador, com uma interface intuitiva que reduz o "no-show" através de uma gestão organizada da agenda.

## Tecnologias
* **Frontend:** React (Vite) ou Vue.js
* **Backend:** Node.js com Express
* **Banco de Dados:** PostgreSQL (via Prisma ORM para facilitar a integração)
* **Agentes de IA:** Gemini (Google), Cursor e Claude Code (para geração e revisão de código)

## Histórias de Usuário
1. Como cliente, quero visualizar a lista de serviços (corte, barba) para saber os preços.
2. Como cliente, quero escolher um barbeiro específico para manter o meu estilo habitual.
3. Como cliente, quero selecionar data e hora em um calendário para agendar meu serviço.
4. Como cliente, quero ver meus agendamentos futuros para me organizar.
5. Como barbeiro, quero visualizar minha agenda do dia para gerenciar meu tempo.
6. Como dono da barbearia, quero cadastrar novos serviços e preços para atualizar o catálogo.
7. Como dono da barbearia, quero cancelar ou remarcar horários em caso de imprevistos.
8. Como administrador, quero ver um resumo de quantos agendamentos foram feitos no mês.
